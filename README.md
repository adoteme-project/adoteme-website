
# Sistema de Adoção de Animais para ONGs

Este projeto é uma aplicação desenvolvida para ajudar ONGs de proteção animal a divulgarem seus animais disponíveis para adoção. A plataforma permite que usuários visualizem informações detalhadas dos pets, filtrem por características específicas (como tamanho, sexo e espécie), e entrem em contato para adotar. O sistema também oferece uma área administrativa para que as ONGs cadastrem e gerenciem os animais.
